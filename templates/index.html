❓ Help & Guide

Learn how to use every feature of Options Copilot

📊 Dashboard

  • Market Overview — See VIX level, SPY price, and overall market sentiment at a glance
  • Sector Sentiment — 11 sectors with daily/weekly/monthly performance and bullish/bearish signals
  • Click any sector card — Expands to show ETF details and leading tickers with live quotes
  • VIX Interpretation — Under 15 = low volatility, 15-25 = normal, 25+ = elevated fear

💼 Portfolio

  • Real Portfolio tab — Shows your actual Public.com holdings when API credentials are configured
  • Paper Portfolio tab — Shows your paper trading positions for practice
  • Without API key — Displays "Connect your account" message with setup instructions
  • Click any position — Opens full technical analysis modal for that underlying
  • Position size % — Shows what percentage of your portfolio each position represents

📈 Trading

  • Ticker Search — Enter any symbol to get full technical analysis (RSI, MACD, ATR, ADX, trend, regime)
  • Sector Quick-Pick — Click Tech, Finance, Healthcare, etc. to see popular tickers in that sector
  • Timeline Selector — Choose chart period: 5D, 1M, 3M, 6M, or 1Y
  • DTE Quick-Picks — Click 3D, 1W, 1M, etc. to jump to the nearest expiration
  • Option Chain — Combined calls/puts view with strike in the center
  • ATM Auto-Scroll — Chain automatically scrolls to the at-the-money strike
  • ITM Highlighting — In-the-money options have a subtle green (calls) or red (puts) background
  • Click any strike — Opens order modal to place a paper or real trade

🔍 Scanner

  • Edge-Based Scoring — Finds high-probability option trades ranked by directional alignment, IV rank, delta, and volume
  • Preset Watchlists — Blue chip, tech, financials, healthcare, energy, or high-volatility stocks
  • Filters — Set minimum volume, open interest, and max DTE to narrow results
  • Signal Types — STRONG_BUY, BUY with win probability and risk/reward ratio
  • Click Trade — Opens order modal for any scanned option

📝 Paper Trading

  • Virtual $10,000 — Practice trading with no real money at risk
  • Quick Trade — Enter symbol, quantity, and optional price to buy or sell
  • Positions Table — See all open positions with real-time P/L
  • Trade History — Complete log of all paper trades
  • Win Rate — Tracks your winning vs losing trades
  • Reset Anytime — Click "Reset Portfolio" to start fresh with $10,000

🤖 Algo Trading

  • Strategy Builder — Create entry/exit rules using dropdowns (RSI, MACD, ATR, trend, regime conditions)
  • Entry Conditions — All conditions must be true (AND logic) to enter a trade
  • Exit Conditions — Any condition can trigger exit (OR logic)
  • Position Sizing — Set % of portfolio per trade, max positions, stop loss, take profit
  • Enabled Toggle — When ON, strategy auto-executes paper trades when conditions are met
  • Live Toggle — ⚠️ When ON, strategy executes REAL orders via Public.com API (requires API key)
  • Backtester — Run any strategy against historical data (3mo to 5y) and see equity curve + metrics
  • Compare Strategies — Select 2-5 strategies to backtest side by side with overlaid equity curves

⚡ SPY Scalper

  • 0DTE Practice Game — Simulated trading for same-day expiry SPY options
  • Live SPY Price — Real-time price with 2-second updates
  • One-Click Trading — BUY CALL or BUY PUT at the nearest ATM strike
  • Position Display — Shows your current option with entry price
  • Close Button — Appears when you have a position, shows current P/L
  • Session P/L — Tracks total profit/loss for this session
  • High Score — Your best session P/L is saved locally

⚙️ Settings

  • API Configuration — Instructions for adding Public.com credentials to enable live trading
  • Indicator Toggles — Enable/disable chart overlays (SMA, Bollinger Bands, etc.)
  • Preferences — All settings are saved locally in your browser

💡 Tips

  • Works without API keys — Paper trading, sentiment analysis, and backtesting are all free
  • Click positions for analysis — Any position in Portfolio or Paper Trading opens a detailed analysis modal
  • Use keyboard — Press Enter in the ticker search to analyze, ESC to close modals
  • Mobile friendly — Sidebar collapses on small screens, tables scroll horizontally

Settings

API Configuration

To enable live trading, add your Public.com API credentials to the .env file:

PUBLIC_COM_SECRET=your_api_secret_here
PUBLIC_COM_ACCOUNT_ID=your_account_id_here

Restart the application after updating credentials.

About Options Copilot

Options Copilot — Public.com Competition Entry

A comprehensive options trading platform with real-time analysis, paper trading, algorithmic strategy building, and live Public.com integration.

Built With

  • Backend: Python, Flask, Public.com SDK
  • Data: yfinance for market data and historical prices
  • Charts: Plotly.js for interactive visualizations
  • Frontend: Vanilla JS, CSS custom properties

Key Features

  • 📊 Real-time market sentiment with 11 sector analysis
  • 💼 Live portfolio integration with Public.com API
  • 📈 Technical analysis: RSI, MACD, ATR, ADX, Bollinger Bands, regime detection
  • 🔍 Edge-based options scanner with win probability scoring
  • 📝 Paper trading with $10,000 virtual portfolio
  • 🤖 Algo strategy builder with backtesting and comparison
  • ⚡ SPY Scalper game for 0DTE practice

✅ Works without API keys — Paper trading, sentiment analysis, technical analysis, backtesting, and the SPY game all work out of the box. Add your Public.com credentials only when you're ready to trade live.
